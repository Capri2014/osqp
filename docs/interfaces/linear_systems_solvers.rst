Linear Systems Solvers
-----------------------
The settings parameter :code:`linsys_solver` defines the solver for the linear system.


+------------------------------------+-------------------------------------+----------------------------------+
| Solver                             | C/C++ Constant                      | Setting value                    |
+====================================+=====================================+==================================+
| SuiteSparse LDL                    | :code:`SUITESPARSE_LDL`             | 0                                |
